# Downloadable APKs

For Galaxy S24 and most modern Android phones, download:

- `kids-play-app-arm64-v8a-release.apk`

Why this file:
- Galaxy S24 uses the ARM64 Android architecture.
- This release APK is much smaller than the debug build and easier to download from GitHub.

Install steps:
1. Download `kids-play-app-arm64-v8a-release.apk` from GitHub.
2. Open the file on your phone.
3. Allow installation from GitHub / browser / files if Android asks.
4. Install and run.

Notes:
- This is a private family test build.
- The app is landscape-first and offline-only.
